Treemap of album's average ranking + search of possible artists

How it works:

- Country-stats API has no data for the artist behind each album. The implemented function searchs each album title in Last.fm API and maps it to the possible artists.

- The resulting data is represented in a TreeMap with levels for the average ranking, meaning red coloured spaces indicate a better ranking.

- Left click on each album will lead to a subdivision of the treemap, where the possible artists are stored.Right click closes the sub-treemap
